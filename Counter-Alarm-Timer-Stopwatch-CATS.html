Counter Alarm Timer Stopwatch - CATS

When the user opens this app, the Stopwatch window will pop up. To start the ‘Stopwatch’, click on ‘Start’. Then record the lap time by clicking on “lap”. This will record the amount of time elapsed and set the timer to 00:00:00:00. The user can also reset the timer by clicking ‘Reset’. Stop the timer by clicking on “Stop”. This will freeze the timer. In order to resume the timer, you can click on ‘Start’.

Select ‘Mode’ > ‘Count Down’. This will open up the countdown counter window. Set the time then click ‘Start’ to start the count down. User can freeze the timer by clicking “Pause”. To unfreeze the timer, click on “Resume”. When the time reaches to 00:00:00:0, you can either play music, shut down the computer or do nothing.

Select ‘Mode’ > ‘Alarm’, this will open up a window where the user has to select the clock type as either 24 hour or 12-hour clock. Then set the alarm time. To turn on the alarm user must make click on ‘Alarm Off’ this will change the font to a red and display ‘Alarm On’. When the alarm starts, the program can either play a music, shut down the computer or do nothing. To hide the window click on ‘Menu’ > ‘Hide’. This will hide the program in the taskbar.

Supports Windows XP, Windows Vista 32/64 bit, Windows 7 32/64 bit, Windows 8 32/64 Bit, Windows 8.1 32/64 Bit, Windows 10 32/64 Bit
