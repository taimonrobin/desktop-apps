Dock Hat Directory Encryption Files DEF

This program will encrypt each and every byte of the selected file or files. To encrypt a single file, open ‘Dock Hat Directory Encryption’ and click on ‘Encryption’ check box > Click ‘Encrypt Multiple Files’. Now if you want to encrypt file name (Basically change file name to a random name) click on 'Encrypt File Names' and if you don't want to encrypt file name then click 'Don't Encrypt File Name' > choose your file by clicking '...' near the text box > click 'Start'. You will get a message that will ask you if you want to encrypt the selected files. Click 'Yes'. The program will encrypt the selected file.

Unfortunately the current version can only encrypt files under 200 MB. If you try encrypt files greater than 200MB then you will get an error message. You can also encrypt multiple files follow the steps mentioned above up to the point where you click '...' to select files > Hold 'ctrl' and click on the files you want to encrypt > then click open. Another way to encrypt files is to copy the full address of the file including the file name and paste it in the textbox. Separate each file with line break. For example

K:\YouTube\Theory\Einstein's Big Idea HD 1080p.mp4
K:\YouTube\[Compilation] Yellow イエロー- Hatsune Miku 初音ミク Project DIVA ドリーミーシアター English lyrics Romaji subtitles.mp4

To decrypt the encrypted file. Click on the ‘Decrypt’ check box > then ‘Decrypt Multiple Files’ select the file or files that you want to decrypt. You can select multiple files by holding on to 'ctrl' button and clicking on the required files. You can also select multiple files by holding on to shift and 'Up' or 'Down' arrow button.

To encrypt all the files in a directory follow these steps. Suppose you want to encrypt ‘C:\Users\ChronusXIII\Downloads\’ directory. Then all the files, files inside folders, it's subfolder and it's subfolder and so on in ‘C:\Users\ChronusXIII\Downloads\’ will be encrypted. Make sure the directory that you are trying to encrypt has files which are less that 200MB else the program will not encrypt. In case files greater than 200MB are found then you will be notified with the number of files that are greater than 200MB including one of those file's location. So to encrypt all files in a directory, click ‘Encrypt’ > ‘Encrypt Folder’ > Depending of whether you want to encrypt the file name or not click ‘Encrypt File Name’ or ‘Don't Encrypt File Name’ > click ‘Start’.

Similarly to decrypt the files in a directory. Click 'Decrypt' > click on 'Decrypt Folder' > click 'Start'.
