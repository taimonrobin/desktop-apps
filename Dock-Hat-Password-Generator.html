Dock Hat Password Generator

Start Dock Hat Password Generator. After installation user will be asked to enter the ‘Master Password’ only once. This is a very important step, user must make sure to set the ‘Master Password’ because if the user loses the ‘Master Password’ there is no way to retrieve it. The user can also set a hint for the password. Click on ‘Hint’ and write something that will help the user to remember the password. Then click on ‘Save Password’. ‘Master Password’ will be needed to retrieve the saved password.

The user can generate a password of any combination and any length. For example, if the user requires a password with only small letter and 12 characters in length. Click on Small Letters and make sure ‘Small Letters’ checkbox is ticked and set the password size to 12. Now, click on generate password. The program will generate the required password and user will not see the password. To see the generated password click on ‘Show Password’ checkbox. For security reasons, it is recommended that user keeps the ‘Show Password’ checkbox unchecked.

Suppose, user needs a password combination containing small letters and capital letters, then tick the checkbox ‘Small Letters’ and ‘Capital Letters’ > Set the ‘Password Size’ > Then click ‘Generate Password’

To use the password or to copy the password simply click ‘Copy To Clipboard’ button. To save the password go to Menu > Save Password. The generated password will be in the text field automatically. Enter the 'Site or Program' field (reference text). In this field enter the name of the site for which user is using the password > Click ‘Save Password’. The password will be saved. Suppose Mr. Crabs generated password is 'krxIi32lbygk' and Mr. Crabs wants to store this password so, he clicked on 'Menu' > 'Save Password As'. Then he entered his ‘Site Name’ as ‘Krusty Krab Restaurant’ > then clicked ‘Save Password’.

A few days later he forgot his password so, to retrieve the password he opened ‘Dock Hat Password Generator’ app and clicked ‘Menu’ > ‘Show Saved Password’ > ‘Search by Site or Program Name’ Then entered the ‘Master Password’. Mr. Crab entered the site name as ‘Krusty Krab Resturant’ > then clicked ‘Search Password’.

A few days later Mr. Crab forgot the name of his restaurant and his generated password. So to retrieve his generated password again. He clicked on ‘Menu’ > ‘Show Saved Password’ > ‘Show All Saved Password’ > Entered ‘Master Password’ and he got all his password saved shown on the screen.
